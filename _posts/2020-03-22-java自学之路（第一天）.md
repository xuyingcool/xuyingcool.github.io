#### 基本的DOS命令

 **打开命令行**（新技巧）：选中一个文件，按住shift + 鼠标右键  选中在此处打开powershell

```
#切换盘符
$ D:

#查看目录下的所有文件
$ dir

#切换目录
$ cd xx
$ cd ..  		#进入上一级

#清楚屏幕
$ cls

#查询ip配置
$ ipconfig

$ ping + 地址

$md xxx   		创建文件夹xx
$md> xx.xx 		创建文件xx.xx

$ del xx.xx   	删除文件
$ rd xx   		产出文件夹
```

####  java的介绍

##### java的特性与优势

简单性、面向对象、可移植（跨平台）、高性能（即时编译）、分布式、动态性（基于反射实现的）、多线程、安全性、健壮性

##### java三大版本

**javaSE**：	标准版

**javaME**：	嵌入式开发

**javaEE**: 		企业版

##### JDK、JRE、JVM

​	**JDK**: java Develpment Kit 

​	**JRE**: Java Runtime Environment

​	**JVM**: Java Virtual Machine  

JDK 包含JRE ， JRE包含JVM。
